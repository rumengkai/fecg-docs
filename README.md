# 代码生成器

## [使用文档](https://temp-static-domain.jd.com/fecg-docs)

## 本组件库使用前，项目需要安装以下依赖

```
react
axios
@ant-design/pro-table
antd
lodash
ahooks
```

## 使用安装

直接全局安装 [fecg](https://www.npmjs.com/package/fecg)

```sh
npm install fecg -g
```

OR

```sh
npm install @jd/fecg -g --registry=http://registry.m.jd.com
```
